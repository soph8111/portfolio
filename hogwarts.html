The Hacked Hogwarts Student List

The Hacked Hogwarts Student List was a school project on third semester. The assignment was to build a system to help the administration at Hogwarts to handle the student list. The project included a lot of JavaScript, designing the interface and manipulating data from a json-file.

As an user you are able to sort, filter, search, expel, select prefects etc. Also, try to click The Golden Snitch in the top left corner.
