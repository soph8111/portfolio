Visual identity X LIIRO DESIGN

This project was the final exam project on 2. semester. In this examination assignment we must develop a multimedia solution for a danish entrepreneur. My group and I worked together with the danish jewelry brand LIIRO DESIGN.

The assignment was about finding a completed visual identity for LIIRO which included a new website (using WordPress and WP REST API) to promote the clients products and communication- and SoMe strategi with our own produced content.

My main role was creating the website in WordPress and to fetch the dynamic data via JavaScript. I also participated in the research, processing the analyzes, (content) idea thinking, taking pictures and designing the communication cards. Setting up the report was also my job.
